CSS attributes

Font and paragraph properties

font-family

specifies one of the following values as comma separated list:

family-name
Any of the available font families installed in the system. For example, Times, Helvetica, Zapf-Chancery, Western, or Courier.
generic-name
Any of the following font families: serif, sans-serif, cursive, fantasy, or monospace.

Example:

body {
  font-family: "Gill Sans Extrabold", Helvetica, sans-serif;
}

font-size

Size of the font, specifies one of the following values:

named-font-size
Set of keywords that indicate predefined font sizes. Scale according to the user’s font setting preferences. Possible values include the following: xx-small, x-small, small, medium, large, x-large, xx-large;
relative-size
Set of keywords that are interpreted as relative to the font size of the parent object. Possible values include the following: larger, smaller;
length
Length Units value;
percentage
Integer, followed by a percent (%). The value is a percentage of the parent object’s font size;

Example:

/* <length> values */
font-size: 12px;
font-size: 0.8em;

/* <percentage> values */
font-size: 80%;

Length Units

Floating-point number, followed by an absolute units designator (cm, mm, in, pt, pc, or px) or a relative units designator (em or ex). For more information about the supported length units, see the CSS Length Units Reference.
