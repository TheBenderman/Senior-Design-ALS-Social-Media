Error object

Error object, is thrown by script runtime when problem is detected.

Constants

N/A

Properties

name
- string, read-only, equals to "Error" for now.
message
- string, read-only, error message.
stackTrace
- string, read-only, stack trace of the error.
data
- any, r/w, optional. Additional payload for the error to carry. For example in view.request() the error contains { status: ..., response: ... } object.
