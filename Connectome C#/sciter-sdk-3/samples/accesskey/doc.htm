Format of accesskey attribute supported by Sciter:
  • "{keyname}" - ALT + key
  • "^{keyname}" - CTRL + key or COMMAND + key (on OSX)
  • "!{keyname}" - key ( as it is )

Where {keyname}  is

  • 0 ... 9  -  one of number keys;
  • A ... Z  -  one of char keys as they appear on English keyboard;
  • or one of the following key names:
    • BACK
