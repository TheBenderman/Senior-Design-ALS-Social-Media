Sciter is an embeddable HTML/CSS/script engine that brings Web stack technologies to Desktop UI development:

Suppports most of HTML5 constructs, except of <video> and <audio> for now;

CSS 2.1 in full with some elements of CSS3 (e.g. transitions);

TIScript engine (a.k.a. JavaScript++), feature set is close to ES6;

CSS extended to better support Desktop UI development, e.g. flow and flex units, vertical and horizontal alignment, OS theming;

Rich and extended set of input elements including <richtext> - WYSYWYG HTML editor;

SDK contains examples and libraries/ports of many popular frameworks like +plus is a non-intrusive data binding mechanism similar to AngularJS.
